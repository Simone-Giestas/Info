Esse trabalho faz parte das atividades da disciplina de Linguagem de Programação Aplicada a WEB. todos as páginas foram criadas usando exclusivamente linguagens HTML e CSS.
Conhecer as linguagens HTML e CSS é fundamental nos dias de hoje por vários motivos:
Fundamentos da Web : HTML (Hypertext Markup Language) é uma linguagem usada para criar a estrutura e o conteúdo de páginas da web, enquanto CSS (Cascading Style Sheets) é responsável pelo design e formatação. Ter um entendimento sólido dessas linguagens é essencial para construir e personalizar páginas web.
Desenvolvimento Web : Se você está interessado em se tornar um desenvolvedor web, HTML e CSS são as bases. Eles são uma porta de entrada para outras linguagens e tecnologias web, como JavaScript, PHP.
Controle Criativo : Através do uso de CSS, você pode controlar completamente a aparência de um site. Isso permite que você projete páginas da web de acordo com suas preferências ou conforme seus clientes, tornando o conhecimento de CSS valioso para designers e desenvolvedores.
Acessibilidade : HTML e CSS são fundamentais para criar sites acessíveis, que podem ser usados ​​por pessoas com deficiências. Com um conhecimento sólido dessas linguagens, você pode estruturar o conteúdo de maneira a garantir que todos tenham acesso a ele.
Responsividade : A capacidade de criar sites responsivos, que se ajustam automaticamente a diferentes dispositivos (como desktops, tablets e smartphones), é crucial na era móvel. Isso é progresso através do uso habilidoso de HTML e CSS.
Economia de Recursos : Ter conhecimento em HTML e CSS pode economizar tempo e recursos. Você pode fazer pequenas edições e configurações em seu site sem depender de terceiros, o que é particularmente útil para proprietários de pequenos negócios e empreendedores.
Compreensão Técnica : Mesmo que você não planeje se tornar um desenvolvedor web profissional, entender HTML e CSS ajuda a comunicar suas necessidades de forma mais eficaz com desenvolvedores ou designers, facilitando a colaboração em projetos web.
Mercado de Trabalho : Há uma demanda constante por profissionais com conhecimentos em HTML e CSS. Muitas empresas procuram pessoas que possam gerenciar e atualizar conteúdo em seus sites, ou que exijam habilidades em HTML e CSS. Em resumo, o conhecimento em HTML e CSS é valioso para qualquer pessoa que queira interagir com a web de forma mais eficaz, seja criando, mantendo ou simplesmente compreendendo como os sites funcionam. Essas habilidades são um ativo importante na era digital.
